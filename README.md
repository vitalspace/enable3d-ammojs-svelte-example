## Install & Run

```console
# download this repo
git clone https://github.com/vitalspace/enable3d-ammojs-svelte-example.git

# navigate into the directory
cd enable3d-ammojs-vite-example

# install
npm install

# start
npm run dev

# build
npm run build

# preview
npm run view
```
